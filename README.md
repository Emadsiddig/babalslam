# 🕌 Bab Alslam Website

A modern and responsive Islamic-themed website built using **HTML**, **CSS**, and **JavaScript**.  
The project focuses on clean design, smooth user experience, and responsive layouts suitable for all devices.

## 🌐 Live Demo

🔗 https://emadsiddig.github.io/babalslam/

## 📌 Features

- Fully responsive design
- Clean and modern interface
- Smooth scrolling navigation
- Beautiful sections layout
- Interactive UI elements
- Mobile-friendly design
- Optimized user experience

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/EmadSiddig/babalslam.git
```

Open the project folder:

```bash
cd babalslam
```

Run the project by opening `index.html` in your browser.
